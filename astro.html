Astro Illustrations

Creative Brief: This project is an exploration of implementing real-world textures into illustration.

Role: Visual Designer

Process: Before the illustration can begin, I start with photographing or painting textures, scanning, then minor processing in photoshop.

Key takeaway: The result is a merging of the digital realm with the imperfections of the tangible world.
